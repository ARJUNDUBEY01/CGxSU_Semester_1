# Do-While Loop in C 

## Theory

A **do-while loop** in C executes a block of code **at least once**,
then checks the loop condition.\
If the condition is `true`, it repeats; otherwise, it stops.

**Syntax:**

``` c
do {
    // code
} while(condition);
```

**Real-world analogy:**\
You always take at least one bite of food before deciding whether to
continue eating.

------------------------------------------------------------------------

## Examples with Detailed Explanation

### 1. Print 1 to 5

``` c
int i = 1;
do {
    printf("%d ", i);
    i++;
} while(i <= 5);
```

**Explanation:** - `i` starts from 1. - Prints the value, increments
`i`, then checks if `i <= 5`. - Runs until `i` becomes 6.

------------------------------------------------------------------------

### 2. User input until 0

``` c
int num;
do {
    scanf("%d", &num);
} while(num != 0);
```

**Explanation:** - Takes input from user at least once. - Continues
until the user enters 0.

------------------------------------------------------------------------

### 3. Reverse counting

``` c
int i = 5;
do {
    printf("%d ", i);
    i--;
} while(i >= 1);
```

**Explanation:** - Starts from 5. - Decreases `i` by 1 after printing. -
Stops when `i` drops below 1.

------------------------------------------------------------------------

### 4. Table of 7

``` c
int i = 1;
do {
    printf("7 x %d = %d\n", i, 7*i);
    i++;
} while(i <= 10);
```

**Explanation:** - Prints multiplication table of 7 from 1 to 10. - Uses
`i` as the multiplier.

------------------------------------------------------------------------

### 5. Sum until negative

``` c
int num, sum = 0;
do {
    scanf("%d", &num);
    if(num > 0) sum += num;
} while(num > 0);
printf("Sum = %d", sum);
```

**Explanation:** - Adds positive numbers to `sum`. - Stops when a
negative number is entered.

------------------------------------------------------------------------

### 6. Password check

``` c
int pass;
do {
    printf("Enter password: ");
    scanf("%d", &pass);
} while(pass != 1234);
printf("Access Granted!");
```

**Explanation:** - Keeps asking for password until correct value
(`1234`) is entered.

------------------------------------------------------------------------

### 7. Factorial of 5

``` c
int i = 1, fact = 1;
do {
    fact *= i;
    i++;
} while(i <= 5);
printf("Factorial = %d", fact);
```

**Explanation:** - Multiplies `fact` by `i` each time. - Computes
factorial of 5.

------------------------------------------------------------------------

### 8. Print alphabets

``` c
char ch = 'A';
do {
    printf("%c ", ch);
    ch++;
} while(ch <= 'Z');
```

**Explanation:** - Prints uppercase alphabets A-Z. - Uses ASCII sequence
to loop through characters.

------------------------------------------------------------------------

## Key Points:

-   **Always runs at least once.**
-   Good for scenarios where the body of the loop must execute before
    condition checking.
-   Condition is checked **after** executing the block.
